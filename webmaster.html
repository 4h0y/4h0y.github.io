<script>
document.addEventListener('DOMContentLoaded', function() {
  window.addEventListener('message', function(e) {
    if (!e || !e.data) return;
    var q = document.querySelector('[data-yo="quality"]');
    var t = document.querySelector('[data-yo="translate"]');
    if (e.data.quality && q) {
      q.innerHTML = e.data.quality;
    }
    if (e.data.translate && t) {
      t.innerHTML = e.data.translate;
    }
  });
});
</script>

Плееры фильмов онлайн для сайта о кино

Решили создать онлайн кинотеатр, но не знаете где брать фильмы онлайн?
Ответ: yohoho.cc

Yohoho - это сайт-агрегатор всех открытых и закрытых видео-хостингов на наличие фильмов онлайн. Вы можете подключить сервис Yohoho во все фильмы на сайте (если сайт уже готов и содержит фильмы) или можете встроить коды выше в шаблон сайта и начать добавлять только информацию о фильме в любой CMS, в которой Вы работаете. Yohoho автоматически отследит о каком фильме идет речь на странице и покажет все плееры из разных источников, которые есть в базе. Помимо этого, Вам не нужно следить за выходом нового качества, доверьте это Yohoho. Как только выйдет на одном из видео-хостингов лучшее качество фильма, Yohoho автоматически добавит его на Ваш сайт.

Как поставить онлайн фильмы на DLE

Для того, чтобы поставить плееры на сайт, нужно отредактировать шаблон. Для этого переходим в папку с Вашим шаблоном и ищем файл «fullstory.tpl», он отвечает за отображение полной новости сайта. Открываем файл любым тектовым редактором, к примеру notepad и добавляем следующий код в место, где бы Вы хотели видеть плееры:
<div id="yohoho" data-title="{title}"></div>
<script src="//yohoho.cc/yo.js"></script>
Код {title} отвечает за отображение названия страницы с фильмов. Важно чтобы название было в таком формате: Русское название / Английское название (год) или Русское название / Английское название или Русское название (год), таким образом можно максимально точно определить, о каком фильме идет речь.

Как поставить онлайн фильмы на Ucoz

Чтобы добавить во на все страницы сайта плееры фильмов, требуется зайти в админ-панель сайта на Ucoz. Перейти в раздел «Дизайн» > «Управление дизайном (шаблоны)». Затем откройте шаблон модуля в который Вы добавляете фильмы, это может быть «Новости сайта» > «Страница материала и комментариев к нему», «Блог» > «Страница материала и комментариев к нему», «Каталог статей» > «Страница материала и комментариев к нему» или «Каталог файлов» > «Страница материала и комментариев к нему». Открыв нужный шаблон, вставьте этот код в место, где хотите видеть плееры:
<div id="yohoho" data-title="$ENTRY_TITLE$"></div>
<script src="//yohoho.cc/yo.js"></script>

Как поставить онлайн фильмы на WordPress

Плееры, которые есть в базе Yohoho на WordPress будут отображаться в шаблоне «single.php». Откройте этот файл с помощью любого тектового редактора и вставьте код в то место, где по Вашему плееры будут смотреться максимально практично и удобно:
<div id="yohoho" data-title="<?php single_post_title(); ?>"></div>
<script src="//yohoho.cc/yo.js"></script>

Как вставить плеер на сайт после клика по кнопке?

Если планируется вставка плеера не сразу (основным), а через клик по кнопке (Плеер 2, Плеер 3 и т.п.), то на кнопке должно быть событие data-ahoy="yohoho". Учтите, это довольно важный параметр, так как если плеер загружается на фоне впустую, Вы расходуете не только русурсы Yohoho, но так же и ресурсы своих пользователей. Сайты, которые будут создавать таким образом фоновую нагрузку будут блокироваться, до тех пор, пока не поставят Yohoho на главный экран (основным) или до того, пока не поставят событие data-ahoy="yohoho" на кнопку.
<a href="javascript:void(0)" data-ahoy="yohoho">Плеер 2</a>
<div id="yohoho" data-title="Дэдпул"></div>
<script src="//yohoho.cc/yo.js"></script>

Как показывать плеер на HTML5 теге <video>?

Есть возможность открывать плеер на HTML5 теге <video> вместо <div>.
<video id="yohoho" data-kinopoisk="401522" controls>
  <source src="//yohoho.cc/yo.mp4" type="video/mp4">
</video>
<script src="//yohoho.cc/yo.js"></script>

Работа плеера онлайн фильмов/сериалов на AndroidTV/SmartTV/STB-приставках

Полное управление плеерами с помощью пульта для телевизора. Настроен выбор плееров с помощью цифровой клавиатуры 1-9, а также стрелок навигации. Для активации используйте параметр data-tv="1", по умолчанию он отключен.

Enter/OK - открыть на весь экран;
1-9 - номер плеера (5 ►);
⬆ - предыдущий плеер;
⬇ - следующий плеер;
⬅ - предыдущая страница;
➡ - следующая страница.

Для автоматического воспроизведения используйте параметр data-autoplay="1". Все современные видео-хостинги поддерживают параметр автовоспроизведения, если плеер автоматически не воспроизводится, пишите в поддержку этого плеера с просьбой добавить URL-параметр autoplay=1.
